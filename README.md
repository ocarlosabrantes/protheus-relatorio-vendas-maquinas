## 🚀 Funcionalidades

- Filtros dinâmicos:
  - Período de emissão
  - Intervalo de vendedores
- Consulta SQL com múltiplos JOINs:
  - SD2 (Itens de venda)
  - SF2 (Notas fiscais)
  - SA1 (Clientes)
  - SA3 (Vendedores)
  - SB1 (Produtos)
  - SDB (Números de série)
  - SF4 (Tipo de operação)
- Exportação para Excel via FWMSExcel
- Barra de progresso durante processamento
- Rastreabilidade por número de série
- Classificação por grupo de produto

---

## 🛠️ Tecnologias utilizadas

- ADVPL (TOTVS Protheus)
- SQL embarcado (TopConn)
- FWMSExcel (geração de Excel)

---

## 📊 Dados apresentados

- Empresa
- Data da nota fiscal
- Número do pedido
- Número e série da nota
- Vendedor (código e nome)
- Cliente
- Loja
- Produto
- Descrição
- Quantidade
- Número de série (máquina)
- Grupo do produto
- Descrição do grupo
- Município e estado
- Tipo de operação

---

## 🔎 Contexto de Negócio

Este relatório é especialmente útil para empresas que:

- Comercializam máquinas ou produtos serializados
- Precisam de rastreabilidade de equipamentos
- Realizam auditorias ou pós-venda
- Analisam desempenho comercial por região e vendedor
